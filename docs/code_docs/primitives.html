Primitives are the building blocks of more complex composite classes, and represent simple rendereable instances.

class pText(text, x, y, kwargs={})ÔÉÅ

Instructions to render a string of text

param string text:

String of text to be rendered.

param number x:

The x-coordinates of the string using current positionMode

param number y:

The y-coordinates of the string using current positionMode

param object kwargs={}:

An dict-object containing additional keyword args

class Countdown(x, y, endtime, kwargs={})ÔÉÅ

A countdown timer that allows a callback to be set for when time elapses, with the option to attach a countdown bar or circular timer.

Arguments:

  • x (number()) ‚Äì The x-coordinates of the timer using current positionMode

  • y (number()) ‚Äì The y-coordinates of the timer using current positionMode

  • endtime (number()) ‚Äì The number of seconds after which the timer elapses.

  • kwargs (object()) ‚Äì An dict-object containing additional keyword args

Countdown.setGraphic(graphic, params={})ÔÉÅ

Sets the countdown graphic to either a circular arc that spans from 360deg to 0, or a progress bar. Also allows the user to extend and add their own graphic using p5.js. TODO: instructions on how to create custom graphic

Arguments:

  • graphic (string()) ‚Äì A string denoting the type of graphic to be added. Allows ‚Äúarc‚Äù for arc, ‚Äúbar‚Äù for progress bar, and ‚Äúcustom‚Äù, which creates an object called this.graphic, where the user can extend the draw method.

  • params (object()) ‚Äì Dict containing params for the graphic. Both arc and bar graphics require width (w) and height (h)

Countdown.reset()ÔÉÅ

Reset the timer. Also used to start the timer.

Countdown.onTimeUp()ÔÉÅ

Empty method that can be set to a callback for when the timer elapses. For example:

content.timer.onTimeUp = () => {
    console.log("Time's up!")
    // Reset the timer, creating a loop
    content.timer.reset();
}
